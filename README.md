# DoubleColorScope 🔴🔵

> 双色球组合空间可视化分析平台 — 换个姿势看彩票，虽然看了也中不了。

## 这是什么

一个用 React + ECharts 搭建的双色球数据可视化看板。它不预测下一期号码（废话，能预测我自己早买了），而是从「组合空间」的视角观察历史开奖数据。

传统分析法把每个号码单独拎出来，统计冷热号、遗漏值——听起来很科学，本质和用 Excel 算命差不多。本项目换了个思路：把每期开奖看成一个完整的 6+1 组合，扔进 110 万种可能红球组合的巨大空间里，看看它们到底怎么分布的。

**结论：极其稀疏，毫无规律，完美符合随机性。** 但这不妨碍我们把它画得好看。

## 为什么你要看这个

- 想用数据说服自己「彩票是真随机」→ 看覆盖率：1009 期才覆盖了 0.09%
- 想找到「热号」→ 看频率图：每个号码出现次数差不多
- 想找到「模式」→ 看 PCA 散点图：一团星云，啥也不是
- 想学 ECharts 怎么用 → 10 种图表示例，比官方文档可读
- 单纯想看漂亮的深色科技风仪表盘 → 这个确实有

## 技术栈

| 技术 | 用途 | 选它的原因 |
|------|------|-----------|
| React 18 + TypeScript | 框架 | 不是我用 Vue 就不会写，但 React 确实更适合看板 |
| Apache ECharts | 图表 | 中文文档，热力图开箱即用，不用自己造轮子 |
| Tailwind CSS | 样式 | 写 CSS 像写小作文，Tailwind 像发电报 |
| 纯前端 | 架构 | 没有后端，没有数据库，没有 API，没有烦恼 |

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 生产构建
npm run build

# 如果新增了开奖数据，重新生成预计算文件
node scripts/build-enhanced-data.mjs
```

## 数据来源

- 1009 期真实双色球开奖数据（2019-07-21 ~ 2026-05-14）
- 数据来源：中国福利彩票官网 + 各省福彩中心
- 预处理：组合编号排名、PCA 降维、各项统计指标全部在构建阶段预计算
- 每次新增开奖数据后运行 `build-enhanced-data.mjs` 即可更新

## 图表说明

打开网页后点右上角 `?` 按钮，每个图表都有通俗解释。这里只列名字：

- 组合编号走势图 — 把红球映射成 1~110 万的编号画折线
- 空间分布直方图 — 看历史开奖在组合空间里是不是均匀的（是的）
- 相邻期距离图 — 相邻两期红球有多不一样
- 重号数量分布 — 0 重号比 6 重号常见得多
- PCA 二维散点 — 33 维降成 2 维，漂亮的星云
- 红球热力图 — 每个号码每年出现多少次
- 和值/奇偶比/大小比/三区分布 — 传统分析指标，该有的都有

## 项目结构

```
src/
├── types/          # 类型定义
├── utils/          # 组合数学、统计、PCA、数据生成
├── context/        # 全局状态管理
├── components/     # 通用 UI 组件（球徽章、卡片、详情面板）
├── charts/         # 10 个 ECharts 图表组件
├── pages/          # 4 个 Tab 页面
└── App.tsx         # 主入口
scripts/            # 数据构建脚本
public/data/        # 原始数据和预计算增强数据
```
